<p style="">This is a simple way to read whole ASCII file into std::string in C++ &minus;</p><h2>Algorithm</h2><pre class="result notranslate">Begin
&nbsp; &nbsp;Declare a file a.txt using file object f of ifstream type to perform read operation.
&nbsp; &nbsp;Declare a variable str of string type.
&nbsp; &nbsp;If(f)
&nbsp; &nbsp; &nbsp; Declare another variable ss of ostringstream type.
&nbsp; &nbsp; &nbsp; Call rdbuf() fuction to read data of file object.
&nbsp; &nbsp; &nbsp; Put the data of file object in ss.
&nbsp; &nbsp; &nbsp; Put the string of ss into the str string.
&nbsp; &nbsp;Print the value of str.
End.</pre><h2>Example</h2><pre class="prettyprint notranslate">#include&lt;iostream&gt;
#include&lt;fstream&gt;
#include&lt;sstream&gt;
#include&lt;string&gt;
using namespace std;
int main() {
&nbsp; &nbsp;ifstream f(&quot;a.txt&quot;); //taking file as inputstream
